summer-game v3.0

Текстовая игра-симулятор о том, как быстро проходит лето.
Теперь с циклом, случайными событиями, инвентарём и защитой от ошибок.

ЧТО НОВОГО

- Ты проживаешь 3 летних дня
- Каждый день может случиться случайное событие (дождь, встреча с другом)
- В магазине случайная цена и скидка
- Ты можешь покупать чипсы, они добавляются в инвентарь
- В конце игра оценивает, как ты провёл лето (по количеству покупок)
- Игра не падает, если ввести буквы вместо чисел

КАК ЗАПУСТИТЬ

1. Убедись, что у тебя установлен Python 3.
   Проверить можно командой:
   python --version

2. Скачай файл v3.0 code.py

3. Открой терминал в папке с файлом.

4. Запусти игру:
   python "v3.0 code.py"

   Если не работает, попробуй:
   python3 "v3.0 code.py"

УПРАВЛЕНИЕ

- Вводи имя, возраст и деньги (только числа)
- Отвечай на вопросы (например, "да" или "нет")
- Следи за инвентарём и остатком денег
